# ZSH-Theme
HTB PWNBOX theme for zsh.

### <ins>Dependencies</ins>
[oh-my-zsh](https://ohmyz.sh) should be installed.

____

### <ins>Installation</ins>
```BASH
git clone https://github.com/gh05t-4/ZSH-Theme.git
cd ZSH-Theme/
cp pwn.zsh-theme ~/.oh-my-zsh/themes/
```
<br>

**Edit `.zshrc`**

```
...
ZSH_THEME="pwn"

...
```
